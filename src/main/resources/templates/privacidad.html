Politica de privacidad

En Metamapa valoramos tu privacidad

1. Información que recopilamos

  • Nombre y Apellido (obligatorio para registro de administradores y contribuyentes).
  • Correo electrónico (opcional, solo si el usuario lo desea).
  • Datos de hechos cargados (título, descripción, ubicación y fecha).

    • 2. Uso de la información

    La información recopilada se utiliza únicamente para:

    • Mostrar hechos y colecciones dentro de la plataforma.
    • Permitir la interacción entre contribuyentes y administradores.
    • Gestionar solicitudes de eliminación y validación de hechos.

    3. Almacenamiento y seguridad

    Los datos se almacenan en la base de datos del sistema con el único propósito de ilustrar las funcionalidades del proyecto. No se utilizan con fines comerciales.

    4. Privacidad de menores

    El sistema no está destinado a menores de 13 años. En caso de detectarse información de este tipo, será eliminada.

    5. Contacto

    Si tenés dudas sobre esta política, escribinos a contacto@metamapa.com
